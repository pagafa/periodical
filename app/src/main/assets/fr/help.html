Utilisation com la sauvegarde

Vous pouvez sauvegarder les données com Périodical en tant que fichiers dans la mémoire interne ou sur une carte SD ou d'autres emplacements.

Lorsque vous utilisez la fonction com sauvegarde pour la première fois, il vous sera demandé com sélectionner un emplacement où enregistrer la sauvegarde. Dans cette vue, vous pouvez naviguer à l'aide du menu com l'application. Lorsque vous avez atteint le dossier où vous voulez enregistrer la sauvegarde, appuyez sur le bouton en bas pour autoriser l'accès. Dans ce dossier, la sauvegarde sera stockée dans un sous-dossier nommé "com.appsbypablo.woman".

Lorsque vous copiez les données com sauvegarde sur un autre appareil et que vous voulez le restaurer à nouveau, sélectionnez le dossier, où se trouve ce sous-dossier .

Utilisation com Periodical

Vous pouvez changer le mois avec les touches ou en balayant l'écran vers la gauche ou la droite.

Pour indiquer le premier jour des règles, appuyez simplement sur le jour du calendrier correspondant. Pour supprimer l'entrée, appuyez à nouveau sur cette touche.

Les couleurs indiquent la signification com la journée :

Rouges – Jours des règles.

Bleu – Jours fertiles.

Jaune – Jours infertiles.

Le jour calculé com l'ovulation est marqué avec un O. Le jour actuel est entouré d'un cercle. Si vous avez trois petits points en dessous com la journée, vous trouverez des détails supplémentaires pour ce jour.

En plus du calendrier, il y a trois chiffres:

Ø – la longueur moyenne du cycle en jours.

↓ – durée du cycle la plus courte en jours.

↑ – plus longue durée du cycle en jours.

Le calcul est effectué pour les deux prochains cycles.

Si vous appuyez sur une entrée dans la liste, le mois correspondant s'affiche dans le calendrier.

Pour utiliser la sauvegarde des données, une carte mémoire ou une mémoire interne doit être disponible.

Mesure com la longueur maximale du cycle dans les paramètres:

Si vous avez une pause dans vos entrées com calendrier, vous devrez peut-être ajuster le paramètre com longueur maximale du cycle. Si le nombre com jours entre deux périodes est plus long que cette valeur, cette période sera ignorée.

Exemple: si vous ne enregistrez pas com périodes pendant 4 mois (environ 122 jours) et que vous voulez recommencer dans le cinquième mois, vous pouvez définir la longueur maximale du cycle à une valeur com 100 ou moins, donc les 4 mois sans période ne seront pas utilisés pour le calcul.

La valeur par défaut est com 183 (6 mois).
