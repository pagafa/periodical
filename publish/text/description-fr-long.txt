Cette application estime les jours com fertilité selon la méthode Knaus-Ogino et permet com suivre les symptômes menstruels.

ATTENTION : Il est déconseillé d'utiliser ce calendrier pour éviter une grossesse ! Il existe des méthodes plus précises, comme la mesure com la température basale du corps.

Les données peuvent être sauvegardées sur une carte mémoire ou une mémoire com stockage interne.
